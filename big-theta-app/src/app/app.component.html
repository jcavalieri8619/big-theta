Big Theta API

The Big Theta API is hosted on AWS. The base url is https://r3psss9s0a.execute-api.us-east-1.amazonaws.com/bigtheta/. All endpoints are relative to this path
GET /equationtree
Params:
  • equation (required): Root equation to search for
  • format (required): Format of equation either ascii or latex
Returns: A tree of related equations with the searched for equation at the root
GET /equations
Params:
  • topic (optional): Topic of the equations to retrieve
Returns: A list of all equations in the database filtered by topic if requested
POST /equations
Body:
  • equation (required): Equation to add
  • topic (required): Topic of the equation
  • format (required): Format of equation either ascii or latex
Returns: The equation itself if addition is successful
